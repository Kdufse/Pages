APatch
新的安卓内核根解决方案

Android 内核版本 3.18-6.6的通用根解决方案

APatch 是一个强大的安卓内核级根解决方案，提供比传统用户空间根方案更高的隐藏性和控制能力。通过内核模块和精细的权限控制，重新定义安卓设备的根访问体验。

开始使用 GitHub 上查看 参与文档制作

核心功能与特性

基于内核的

APatch 运行在内核空间，比用户空间根具有更高的隐藏性和控制。直接与内核交互，提供更底层的系统访问能力，同时保持更好的隐蔽性。

根访问控制

只有被允许的应用才能访问或看到用户，其他应用都不知道这一点。细粒度的权限管理确保系统安全，防止恶意应用滥用根权限。

APModule

支持类似Magisk的模块系统。通过模块扩展系统功能，兼容现有的Magisk模块生态，同时提供更稳定可靠的运行环境。

KPModule

支持允许你向内核注入任意代码的模块（提供内核函数 inline-hook 和 syscall-table-hook）。为高级用户提供深度定制能力。

APatch 项目

开源的安卓内核级根解决方案，为开发者和高级用户提供强大的设备控制能力。
